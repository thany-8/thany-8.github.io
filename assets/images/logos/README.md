# Card logos

Drop your company / school logo image files here to replace the text monograms
in the **Experience** and **Education** cards. Until a matching file exists, the
card automatically shows its monogram (e.g. `FR`) instead — nothing breaks.

**Recommended:** a square-ish PNG or SVG with a transparent or white background,
at least 140×140px.

## Expected filenames (referenced in `index.html`)

| Card                            | File                    |
| ------------------------------- | ----------------------- |
| FlyRank.ai                      | `flyrank.png`           |
| CodePath                        | `codepath.png`          |
| ColorStack                      | `colorstack.png`        |
| College of Marin                | `college-of-marin.png`  |
| San Francisco State University  | `sfsu.png`              |

To use a different name or an `.svg`, just update the matching
`<img src="assets/images/logos/…">` in `index.html`.
